# Sequence Detector FSM (1010)

This project implements a 1010 sequence detector using:

- Mealy FSM
- Moore FSM

Files:
- fsm.v           : Mealy sequence detector
- fsm_tb.v        : Mealy testbench
- fsm_mor.v       : Moore sequence detector
- fsm_mor_tb.v    : Moore testbench

Waveforms:
- fsm_mealy_1010.png
- fsm_moore_1010.png

Tools Used:
- Verilog HDL
- Vivado
- XSim
